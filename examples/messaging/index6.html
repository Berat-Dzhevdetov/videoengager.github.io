Advanced Session Manager - UserScript Demo

This page is the target for the "Advanced Session Manager" Tampermonkey UserScript. The UserScript will dynamically add buttons and a video call interface to this page.

How to Use This Demo:

1. Install a UserScript Manager: If you haven't already, install a UserScript manager extension in your browser. Tampermonkey is a popular choice (available for Chrome, Firefox, Edge, Safari, etc.).

2. Install the UserScript: Click the link below to install the "Advanced Session Manager Demo" UserScript. Your UserScript manager should automatically detect it and prompt you for installation.
Install the UserScript
(Replace [YOUR_RAW_LINK_TO_THE_USER_SCRIPT.JS_FILE] with the actual raw link to your .user.js file, e.g., on GitHub.)

3. Configure @match (Important): The provided UserScript is initially configured with // @match https://www.videoengager.com/*. For this demo page to work, you must update the @match directive in the script (via the Tampermonkey dashboard) to include the URL of this current page.
For example, if this page is hosted at https://your-username.github.io/your-repo/tampermonkey-demo.html, you would add or change the line to:
// @match https://your-username.github.io/your-repo/tampermonkey-demo.html
(Ensure you save the changes in Tampermonkey after editing the script.)

4. See it in Action: Once the UserScript is installed, configured for this page's URL, and enabled in Tampermonkey, reload this page. You should see the demo buttons appear (e.g., "Support", "Sales"). The video UI will also be managed by the script.

The content below this point is just part of this simple HTML page. The interactive elements are injected by the UserScript.
